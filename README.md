## Como usar localmente

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/pedro-lucena-nutricionista.git
cd pedro-lucena-nutricionista
```

2. Abra o arquivo `index.html` em um navegador ou inicie um servidor local:
```bash
npx http-server -p 3000
```

3. Acesse `http://localhost:3000`

## Deploy na Vercel

1. Faça push do código para GitHub
2. Vá para [vercel.com](https://vercel.com)
3. Clique em "New Project"
4. Selecione o repositório
5. Clique em "Deploy"

O site será automaticamente publicado na Vercel!

## Customização

### Cores
Edite as variáveis CSS em `styles.css`:
```css
:root {
    --primary-color: #1a7d4e;
    --primary-dark: #0d4a2c;
    --primary-light: #2aad6f;
    /* ... */
}
```

### Conteúdo
Edite os textos diretamente em `index.html`

### Contato
Atualize as informações de contato na seção `#contato`:
- Telefone
- Email
- Localização
- Links de redes sociais
